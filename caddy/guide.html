캐디 수입장부 사용법

라운딩 수입을 간편하게 기록하고 통계를 확인하세요

📋 목차

  1. 수입 기록하기
  2. 내역 확인·수정·삭제
  3. 월간·연간 통계 보기
  4. CSV 다운로드
  5. 홈화면에 설치하기
  6. 자주 묻는 질문
STEP 1

수입 기록하기

하단의 「수입 기록하기 ✏️」 버튼을 눌러 당일 라운딩 수입을 기록할 수 있습니다.

입력 항목

📅 날짜 라운딩 날짜 선택 (기본값: 오늘)
⛳ 근무 홀수 라운딩한 홀 수 입력
💗 캐디피 기본 캐디피 금액 (원)
🧡 팁 팁 금액 (선택 입력)
📝 메모 특이사항 메모 (선택 입력)
기록 저장하기
모든 항목을 입력한 뒤 저장 버튼을 누르면 기록이 저장됩니다. 캐디피와 팁은 자동으로 합산되어 총 수입으로 표시됩니다.
팁과 메모는 선택 항목이에요. 날짜, 홀수, 캐디피만 입력해도 저장됩니다.
STEP 2

내역 확인·수정·삭제

저장된 수입 내역은 홈 화면에서 날짜별로 확인할 수 있습니다.

내역 확인
홈 화면에서 해당 월의 기록 목록이 최신순으로 표시됩니다. 각 카드에 날짜, 캐디피, 팁, 합계가 표시됩니다.
수정하기 ✏️
내역 카드를 터치하면 상세 정보가 펼쳐집니다. 「수정하기」 버튼을 눌러 금액이나 메모를 수정할 수 있습니다.
삭제하기
「삭제」 버튼을 누르면 확인 팝업이 뜹니다. 확인을 누르면 해당 기록이 영구 삭제됩니다.
삭제한 기록은 복구할 수 없으니 신중하게 삭제해주세요.
STEP 3

월간·연간 통계 보기

캐디 수입을 월별, 연도별로 한눈에 파악할 수 있습니다.

📊 월간 통계

상단의 월 선택 화살표(◀ ▶)로 원하는 달을 선택하면 해당 월의 통계가 자동으로 계산됩니다.

항목 설명
총 수입 해당 월의 캐디피 + 팁 합계
캐디피 / 팁 캐디피와 팁을 각각 분리해서 확인
근무 일수 해당 월 라운딩 횟수
총 홀수 해당 월 전체 홀 수 합계
일 평균 수입 총 수입 ÷ 근무 일수

📈 연간 통계

하단 네비게이션의 「연간 통계」 탭을 누르면 해당 연도의 월별 수입 추이와 연간 합계를 확인할 수 있습니다.

통계는 기록을 저장할 때마다 자동으로 업데이트됩니다. 별도 조작이 필요 없어요!
STEP 4

CSV 다운로드

기록한 수입 내역을 CSV 파일로 다운로드하여 엑셀이나 구글 시트에서 활용할 수 있습니다.

다운로드 방법
통계 화면 또는 설정에서 「CSV 다운로드」 버튼을 누르면 전체 기록이 CSV 파일로 저장됩니다.
CSV 파일 내용
날짜, 근무 홀수, 캐디피, 합계가 포함됩니다. 엑셀에서 열어 추가 분석이나 세금 신고 자료로 활용할 수 있어요.
CSV 파일명은 「캐디수입_YYYY.csv」 형식으로 자동 생성됩니다.
STEP 5

홈화면에 설치하기

앱스토어 설치 없이 홈화면에 추가하면 앱처럼 바로 실행할 수 있습니다.

🤖
안드로이드 (Chrome)
① 크롬에서 사이트 접속
② 우측 상단 ⋮ 메뉴
③ 「홈 화면에 추가」 터치
④ 추가 확인
🍎
아이폰 (Safari)
① 사파리에서 사이트 접속
② 하단 공유 버튼 (□↑) 터치
③ 「홈 화면에 추가」 선택
④ 추가 확인
홈화면에서 실행하면 주소창 없이 앱처럼 깔끔하게 사용할 수 있어요.
FAQ

자주 묻는 질문

Q. 데이터가 날아갔어요!
데이터는 브라우저에 저장됩니다. 브라우저 캐시/데이터를 삭제하면 초기화될 수 있어요. 시크릿(비공개) 모드에서는 저장이 안 됩니다.
Q. 다른 폰에서도 볼 수 있나요?
현재는 기기별 로컬 저장 방식이라, 기기마다 따로 기록해야 합니다. 중요한 데이터는 CSV로 백업해두세요.
Q. 다른 사람이 내 기록을 볼 수 있나요?
아니요! 데이터는 내 기기 브라우저에만 저장되고, 서버로 전송되지 않습니다. 같은 링크로 접속해도 각자 기기에 별도로 저장됩니다.
Q. PC에서도 사용 가능한가요?
네, PC 브라우저에서도 동일하게 사용할 수 있습니다. 다만 모바일에 최적화된 디자인이에요.
